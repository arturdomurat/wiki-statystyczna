Mediacja i moderacja — jak testować?

Kiedy tego potrzebujesz

Chcesz sprawdzić jak zmienna X wpływa na Y: czy przez zmienną pośredniczącą M (mediacja — mechanizm), czy efekt X→Y zależy od wartości innej zmiennej W (moderacja — warunek brzegowy).

Mediacja (X → M → Y)

Czym jest

X wpływa na Y przez M. Np.: trening uważności (X) → zmniejszenie ruminacji (M) → obniżenie lęku (Y). Ruminacja jest mediatorem — mechanizmem, przez który trening działa.

Efekty

  • a: X → M (wpływ X na mediator)
  • b: M → Y (wpływ mediatora na Y, kontrolując X)
  • c’: X → Y (efekt bezpośredni, kontrolując M)
  • c: X → Y (efekt całkowity, bez kontroli M)
  • a × b: efekt pośredni (kluczowy!) — c = c’ + a×b

Podejście klasyczne — Baron i Kenny (1986)

Historycznie mediację testowano w 4 krokach (Baron & Kenny, 1986): wykaż istotne ścieżki a, b i c, a następnie pokaż, że po wprowadzeniu mediatora efekt całkowity c maleje (do c’). Dawniej stosowano również test Sobela, który sprawdzał istotność efektu pośredniego a × b.

To podejście jest dziś uznawane za niewystarczające z dwóch powodów: (a) wymaga istotności efektu całkowitego c, ale mediacja może zachodzić bez istotnego c (inconsistent mediation — gdy efekt bezpośredni i pośredni mają przeciwne znaki); (b) test Sobela zakłada, że iloczyn a × b ma rozkład normalny, co w praktyce jest nierealistyczne, szczególnie przy mniejszych próbach.

Współczesne podejście — PROCESS (Hayes)

Makro PROCESS (Andrew F. Hayes) — aktualny standard w psychologii:

  • Model 4: prosta mediacja
  • Bootstrap (≥ 5000 prób) dla efektu pośredniego (a × b)
  • Jeśli bootstrapowy CI nie zawiera 0 → mediacja istotna
  • Bootstrap nie wymaga założenia o normalności rozkładu efektu pośredniego — dlatego zastąpił test Sobela

PROCESS jest dostępne dla: SPSS, SAS, R (pakiet processR lub oryginalne makro).

Moderacja (X × W → Y)

Czym jest

Efekt X na Y zależy od wartości W. Np.: efekt terapii (X) na poprawę (Y) zależy od nasilenia objawów wyjściowych (W). W jest moderatorem.

Testowanie

  • PROCESS Model 1 lub regresja z interakcją
  • W modelu regresji: Y = b0 + b1X + b2W + b3(X × W) + e
  • Istotne b3 → moderacja potwierdzona

Probing interakcji

Po potwierdzeniu moderacji: analiza efektów prostych — jaki jest efekt X na Y przy różnych wartościach W?

  • Pick-a-point: efekt X przy W = M−SD, M, M+SD
  • Johnson-Neyman: dokładne wartości W, przy których efekt X staje się istotny/nieistotny

Centrowanie predyktorów w moderacji

Przed utworzeniem interakcji (X × W) zmienne ciągłe X i W należy wycentrować — odjąć od każdej wartości średnią danej zmiennej. Dlaczego?

  • Interpretacja efektów głównych: bez centrowania b1 (efekt X) opisuje wpływ X, gdy W = 0 — a zero może nie mieć sensu (np. wiek = 0). Po wycentrowaniu b1 opisuje efekt X przy średnim poziomie W, co jest interpretowalne.
  • Współliniowość: iloczyn X × W jest silnie skorelowany z X i W, gdy zmienne nie są wycentrowane. Centrowanie redukuje tę współliniowość (choć nie wpływa na istotność samej interakcji).
  • Uwaga: centrowanie nie zmienia testu interakcji (b3) ani wyników analizy efektów prostych — zmienia jedynie interpretację b1 i b2.

Szczegóły techniczne i kontekst → hasło Centrowanie zmiennych.

Modele złożone

  • Moderowana mediacja (conditional indirect effect): efekt pośredni a×b zależy od moderatora (PROCESS modele 7, 8, 14, 15)
  • Mediowana moderacja: interakcja X×W wpływa na Y przez M
  • Przy złożonych modelach rozważ SEM zamiast PROCESS

Założenia

  • Mediacja/moderacja: te same co regresja liniowa (liniowość, normalność reszt, homoscedastyczność, brak współliniowości)
  • Przyczynowość: mediacja zakłada kierunek X → M → Y. Dane przekrojowe nie dowodzą przyczynowości! Tylko eksperyment lub dane podłużne mogą wspierać interpretację przyczynową.

Raportowanie (APA 7)

Mediacja

„Efekt pośredni treningu uważności na lęk przez ruminację był istotny, b = −2.34, SE = 0.78, 95% bootstrapowy CI [−4.12, −1.02]. Efekt bezpośredni (c’) nie był istotny, b = −0.56, p = .412, co wskazuje na pełną mediację."

Moderacja

„Stwierdzono istotną interakcję terapia × nasilenie objawów, b = 0.45, SE = 0.18, t(96) = 2.50, p = .014. Analiza efektów prostych (Johnson-Neyman) wykazała, że efekt terapii był istotny przy nasileniu objawów powyżej 3.2 punktu."

Częste błędy

  1. Wniosek o przyczynowości z danych przekrojowych. „X powoduje Y przez M" — tylko jeśli masz dane podłużne lub eksperyment. Na danych przekrojowych poprawnie: „wyniki są spójne z modelem mediacji."

  2. Stosowanie testu Sobela zamiast bootstrapu. Test Sobela zakłada normalność efektu pośredniego — bootstrap nie. Bootstrap jest standardem.

  3. Mylenie mediacji z moderacją. Mediacja = mechanizm (jak). Moderacja = warunek (kiedy/dla kogo).

  4. Brak centrowania predyktorów — patrz niżej.

  5. Ignorowanie wielkości efektu pośredniego. Raportuj nie tylko istotność, ale i wielkość efektu (np. κ² lub proporcja mediacji).
