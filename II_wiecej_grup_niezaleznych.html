Więcej niż dwie grupy niezależne — który test?

Kiedy tego potrzebujesz

Masz trzy lub więcej grup niezależnych i jedną zmienną zależną. Np. porównujesz wyniki trzech metod terapeutycznych albo poziom stresu w czterech grupach zawodowych. Chcesz sprawdzić, czy grupy różnią się między sobą.

Kontekst próby

(a) Analiza mocy + wymagane n osiągnięte: ścieżka parametryczna (ANOVA).

(b) Bez analizy mocy: sprawdź n w grupach, rozkłady, wariancje → algorytm poniżej.

(c) Małe próby kliniczne: Kruskal-Wallis. Przy bardzo małych n (< 5 w grupie) moc będzie niska — raportuj to.

Algorytm decyzyjny

Uwaga: ANOVA jest testem omnibusowym — mówi CZY gdzieś są różnice, nie GDZIE. Do odpowiedzi na pytanie „które grupy się różnią" potrzebne są testy post hoc.

1. Zmienna przedziałowa, normalność OK (lub duże n), wariancje homogeniczne

→ Jednoczynnikowa ANOVA (one-way ANOVA)

Post hoc:

  • Tukey HSD — zalecany domyślnie, kontroluje family-wise error rate, najlepsza moc przy równych n
  • Bonferroni — bardziej konserwatywny, ale elastyczny (działa przy nierównych n)
  • Scheffé — najbardziej konserwatywny, zalecany gdy chcesz testować dowolne kontrasty (nie tylko parami)

2. Zmienna przedziałowa, normalność OK (lub duże n), wariancje HETEROGENICZNE

→ Welch ANOVA (w R: oneway.test())

Post hoc: Games-Howell — nie wymaga homogeniczności wariancji ani równych n.

3. Zmienna przedziałowa, małe próby, brak normalności

→ Test Kruskal-Wallisa (nieparametryczny odpowiednik ANOVA)

Post hoc: test Dunna z poprawką Bonferroniego lub Holma.

4. Zmienna porządkowa

→ Test Kruskal-Wallisa

Założenia

ANOVA jednoczynnikowa

  1. Niezależność obserwacji — osoby w różnych grupach niezależne od siebie.
  2. Skala co najmniej przedziałowa.
  3. Normalność rozkładów w każdej grupie — przy małych n. Przy dużych CTG łagodzi.
  4. Homogeniczność wariancji — test Levene’a. Jeśli niespełnione → Welch ANOVA.

ANOVA jest dość odporna na naruszenia normalności, szczególnie przy równolicznych grupach.

Kruskal-Wallis

  1. Niezależność obserwacji.
  2. Skala co najmniej porządkowa.
  3. Podobne kształty rozkładów w grupach (jeśli chcemy interpretować jako test median).

Raportowanie (APA 7)

ANOVA

„Stwierdzono istotny efekt metody terapii na poziom lęku, F(2, 87) = 5.12, p = .008, η²p = .11. Testy post hoc Tukey’ego wykazały, że grupa CBT (M = 38.2, SD = 8.1) miała istotnie niższy lęk niż grupa kontrolna (M = 45.6, SD = 9.3), p = .006, d = 0.85, ale nie różniła się istotnie od grupy psychodynamicznej (M = 41.1, SD = 8.7), p = .342."

Welch ANOVA

„Ze względu na heterogeniczność wariancji (test Levene’a: p = .018) zastosowano ANOVA Welcha, F(2, 54.3) = 4.78, p = .012, z testami post hoc Games-Howella."

Kruskal-Wallis

„Test Kruskal-Wallisa wykazał istotne różnice między grupami, H(2) = 9.34, p = .009. Testy post hoc Dunna z poprawką Holma wykazały…"

Częste błędy

  1. Wielokrotne testy t zamiast ANOVA. Porównywanie 3 grup trzema testami t (A vs B, A vs C, B vs C) powoduje inflację błędu I rodzaju. ANOVA + post hoc kontrolują to.

  2. Raportowanie samej ANOVA bez post hoc. Istotne F mówi „coś się różni", ale nie mówi co od czego. Zawsze podaj wyniki post hoc.

  3. Tukey HSD przy heterogenicznych wariancjach. Tukey zakłada homogeniczność — jeśli Levene istotny, użyj Games-Howella.

  4. Brak η²p lub innej miary efektu. ANOVA bez wielkości efektu jest niekompletna.

  5. Stosowanie wielu porównań bez poprawki. Testy post hoc powinny korygować na porównania wielokrotne.
