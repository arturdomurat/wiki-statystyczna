Więcej niż dwie grupy zależne (pomiary powtarzane) — który test?

Kiedy tego potrzebujesz

Masz trzy lub więcej pomiarów tej samej zmiennej u tych samych osób (np. pre, post, follow-up) lub kilka warunków w schemacie within-subjects. Chcesz sprawdzić, czy zachodzi zmiana.

Kontekst próby

(a) Analiza mocy + n osiągnięte: ANOVA z powtarzanymi pomiarami. Pomiary powtarzane mają większą moc (eliminujesz zmienność międzyosobową).

(b) Bez analizy mocy: sprawdź normalność i sferyczność → algorytm poniżej.

(c) Małe próby: Friedman. Przy bardzo małych n moc niska, ale ANOVA z powtarzanymi pomiarami jest dość odporna.

Algorytm decyzyjny

1. Zmienna przedziałowa, normalność OK, sferyczność spełniona

→ ANOVA z powtarzanymi pomiarami (repeated measures ANOVA)

Post hoc: porównania parami z poprawką Bonferroniego lub Holma.

2. Zmienna przedziałowa, normalność OK, BRAK sferyczności

→ ANOVA z korektą:

  • Greenhouse-Geisser (G-G): gdy ε < .75 (poważne naruszenie). Bardziej konserwatywna.
  • Huynh-Feldt (H-F): gdy ε ≥ .75 (umiarkowane naruszenie). Mniej konserwatywna.

Oba podejścia korygują stopnie swobody (zmniejszają df), co daje bardziej liberalny próg p.

Alternatywnie: wielopoziomowa ANOVA (MANOVA podejście) — nie wymaga sferyczności w ogóle.

3. Zmienna przedziałowa, brak normalności / zmienna porządkowa

→ Test Friedmana (nieparametryczny odpowiednik ANOVA z powtarzanymi pomiarami)

Post hoc: test Wilcoxona parami z poprawką Bonferroniego.

Sferyczność — czym jest i jak sprawdzić

Sferyczność to założenie specyficzne dla pomiarów powtarzanych (≥ 3 warunki). Oznacza, że wariancje różnic między wszystkimi parami warunków są równe.

Np. przy 3 pomiarach (A, B, C): Var(A−B) ≈ Var(A−C) ≈ Var(B−C).

Nie ma odpowiednika w schematach between-subjects (tam odpowiednikiem jest homogeniczność wariancji).

Test Mauchly’ego: testuje H0: sferyczność spełniona.

  • p > .05 → brak podstaw do odrzucenia sferyczności → ANOVA bez korekty
  • p ≤ .05 → sferyczność naruszona → zastosuj korektę G-G lub H-F

Parametr ε (epsilon): mierzy stopień naruszenia sferyczności (1 = idealna, niższe = gorzej).

Uwaga: przy 2 pomiarach sferyczność jest zawsze spełniona automatycznie (jest tylko jedna para różnic). Dlatego sferyczność ma znaczenie dopiero od 3 warunków wzwyż.

Założenia

ANOVA z powtarzanymi pomiarami

  1. Zależność obserwacji — te same osoby we wszystkich warunkach.
  2. Skala co najmniej przedziałowa.
  3. Normalność zmiennej w każdym warunku — przy małych n.
  4. Sferyczność — wariancje różnic między parami warunków równe. Jeśli niespełnione → korekta G-G/H-F.

Test Friedmana

  1. Zależność obserwacji.
  2. Skala co najmniej porządkowa.

Raportowanie (APA 7)

ANOVA z powtarzanymi pomiarami (sferyczność OK)

„Stwierdzono istotny efekt czasu na poziom lęku, F(2, 58) = 8.34, p < .001, η²p = .22."

ANOVA z korektą G-G

„Test Mauchly’ego wykazał naruszenie sferyczności, χ²(2) = 8.12, p = .017, ε = .72. Po korekcie Greenhouse-Geissera efekt czasu pozostał istotny, F(1.44, 41.76) = 7.21, p = .005, η²p = .20."

Uwaga: df po korekcie są niecałkowite (np. 1.44 zamiast 2).

Friedman

„Test Friedmana wykazał istotne różnice między pomiarami, χ²(2) = 12.45, p = .002."

Częste błędy

  1. Ignorowanie sferyczności. Niesprawdzenie testu Mauchly’ego i raportowanie ANOVA bez korekty → zawyżony błąd I rodzaju.

  2. Stosowanie korekty G-G gdy sferyczność spełniona. Niepotrzebna korekta zmniejsza moc.

  3. Wielokrotne testy t dla prób zależnych zamiast ANOVA. Porównywanie pre-post, pre-follow-up, post-follow-up trzema testami t → inflacja α. ANOVA + post hoc kontrolują to.

  4. Ignorowanie efektu kolejności w schemacie within-subjects bez counterbalancingu.

  5. Nieraportowanie ε (epsilon) przy korekcie — czytelnik nie wie, jak poważne było naruszenie.
