SPSS — najczęstsze operacje

Kiedy tego potrzebujesz

Używasz SPSS i potrzebujesz szybkiego przypomnienia: jak przeprowadzić typowe analizy krok po kroku. Albo masz dane w SPSS i nie chcesz przechodzić na inne narzędzie.

Co to jest SPSS

SPSS (Statistical Package for the Social Sciences) to komercyjne oprogramowanie statystyczne, standard w naukach społecznych od lat 80. Interfejs: arkusz danych + menu + okna dialogowe. Działa na zasadzie „klik → wynik". Ma też język skryptowy Syntax.

Dlaczego SPSS jest nadal popularny

  • Szeroko znany — większość psychologów uczyła się na SPSS.
  • Intuicyjny interfejs — menu-driven, nie wymaga programowania.
  • Licencja instytucjonalna — większość uczelni ma dostęp.
  • Dobra dokumentacja — podręczniki (np. Pallant, Field) pisane pod SPSS.

Ograniczenia SPSS

  • Płatny — bez licencji instytucjonalnej kosztowny.
  • Mniej elastyczny niż R — niestandardowe analizy wymagają Syntax lub wtyczek.
  • Bayes Factor — nie obsługuje natywnie (potrzebna wtyczka lub JASP).
  • Brak nowoczesnych metod — modele mieszane (MIXED) są, ale SEM wymaga AMOS (osobny program).
  • Reprodukowalność — klikanie w menu nie zostawia śladu. Zawsze zapisuj Syntax!

Najczęstsze analizy — ścieżka w menu

Statystyki opisowe

Analyze → Descriptive Statistics → Descriptives (lub Frequencies, Explore)

Test t dla prób niezależnych

Analyze → Compare Means → Independent-Samples T Test

  • Test Variables: DV
  • Grouping Variable: IV (Define Groups: 1 vs. 2)
  • SPSS domyślnie podaje zarówno equal variance (Student) jak i Levene + Welch → raportuj Welch

Test t dla prób zależnych

Analyze → Compare Means → Paired-Samples T Test

ANOVA jednoczynnikowa

Analyze → Compare Means → One-Way ANOVA

  • Post Hoc: zaznacz Tukey HSD (lub Games-Howell przy nierównych wariancjach)
  • Options: Descriptive, Homogeneity of variance test, Welch

ANOVA z powtórzeniami

Analyze → General Linear Model → Repeated Measures

  • Definiuj czynnik within (nazwa + liczba poziomów)
  • Przenosi zmienne do poziomów
  • Options: Descriptive statistics, Estimates of effect size
  • Uwaga na sferyczność: sprawdź test Mauchly’ego, raportuj G-G lub H-F przy naruszeniu

ANOVA czynnikowa

Analyze → General Linear Model → Univariate

  • Dependent Variable: DV
  • Fixed Factor(s): IV1, IV2
  • Options: Descriptive statistics, Estimates of effect size, Homogeneity tests
  • Post Hoc: dla efektów głównych

Korelacja

Analyze → Correlate → Bivariate

  • Zaznacz Pearson i/lub Spearman
  • Flag significant correlations

Regresja liniowa

Analyze → Regression → Linear

  • Dependent: DV
  • Independent(s): predyktory
  • Method: Enter (preferowana) / Stepwise (ostrożnie!)
  • Statistics: Collinearity diagnostics, Durbin-Watson
  • Plots: ZRESID vs. ZPRED (homoscedastyczność), Normal P-P Plot (normalność reszt)

Regresja logistyczna

Analyze → Regression → Binary Logistic

  • Dependent: DV (dychotomiczna, kodowana 0/1)
  • Covariates: predyktory
  • Options: Classification plots, Hosmer-Lemeshow, CI for exp(B)

χ² (chi-kwadrat)

Analyze → Descriptive Statistics → Crosstabs

  • Statistics: Chi-square, Phi and Cramer’s V
  • Cells: Expected (zaznacz, żeby sprawdzić fe ≥ 5)
  • Exact: Fisher’s exact test (przy małych fe)

Rzetelność

Analyze → Scale → Reliability Analysis

  • Model: Alpha (domyślny)
  • Statistics: Scale if item deleted, Inter-item correlations
  • UWAGA: SPSS nie liczy ω McDonalda — do tego potrzebujesz R lub JASP

EFA

Analyze → Dimension Reduction → Factor

  • Extraction: Principal Axis Factoring (NIE PCA, jeśli chcesz EFA)
  • Rotation: Oblimin (Direct Oblimin) przy skorelowanych czynnikach; Varimax jeśli zakładasz brak korelacji
  • Options: Suppress small coefficients (< .30)

Syntax — dlaczego warto

Zapisuj Syntax dla każdej analizy. Menu SPSS nie zostawia śladu — za rok nie będziesz pamiętał, co kliknąłeś. Syntax zapewnia reprodukowalność.

Jak: po ustawieniu opcji w oknie dialogowym kliknij Paste zamiast OK → komenda pojawi się w oknie Syntax → Run.

Przykład Syntax:

T-TEST GROUPS=grupa(1 2)
  /MISSING=ANALYSIS
  /VARIABLES=wynik
  /CRITERIA=CI(.95).

Częste błędy

  1. Klikanie OK zamiast Paste. Bez Syntax nie masz dokumentacji tego, co zrobiłeś. Zawsze Paste → Save Syntax.

  2. Raportowanie testu Studenta zamiast Welcha. SPSS domyślnie podaje oba. Raportuj Welch (Equal variances not assumed) — jest bezpieczniejszy.

  3. Używanie PCA zamiast EFA. W menu Factor domyślna metoda ekstrakcji to PCA — zmień na PAF, jeśli chcesz eksploracyjną analizę czynnikową.

  4. Ignorowanie kodowania zmiennych. SPSS wymaga numerycznych kodów (1, 2, 3) z etykietami wartości. Sprawdź Variable View: Type, Values, Measure.

  5. Brak sprawdzenia założeń. SPSS nie sprawdza ich automatycznie — musisz sam zaznaczyć odpowiednie opcje (Levene, normalność reszt, VIF).
