Wiki Statystyczna › Porównywanie grup

Dwie grupy zależne (pomiary powtarzane) — który test?

Kiedy tego potrzebujesz

Masz pomiar tej samej zmiennej u tych samych osób w dwóch momentach (pre-post) lub w dwóch warunkach (within-subjects). Np. mierzysz lęk przed i po terapii u tych samych pacjentów. Chcesz sprawdzić, czy jest istotna różnica.

Kontekst próby

(a) Analizę mocy wykonano, n osiągnięte Próby zależne mają większą moc niż niezależne (bo eliminujesz zmienność międzyosobową). Zwykle ścieżka parametryczna.

(b) Bez analizy mocy Sprawdź n i normalność różnic → algorytm poniżej.

(c) Małe próby kliniczne Test Wilcoxona signed-rank. Przy n < 10 rozważ dokładne wartości p (exact test).

Algorytm decyzyjny

Kluczowa uwaga: testujemy normalność różnic (di = x1i − x2i), nie samych zmiennych!

1. Zmienna przedziałowa/ilorazowa

1a. Duża próba (n ≥ 30) — rozkłady dowolne

→ Test t dla prób zależnych (paired-samples t-test)

CTG działa na różnicach — normalność zmiennych ani różnic nie jest wymagana.

1b. Mała próba (n < 30), rozkład różnic normalny

→ Test t dla prób zależnych

Normalność różnic sprawdź testem Shapiro-Wilka (na zmiennej di = x1 − x2).

1c. Mała próba (n < 30), brak normalności różnic

→ Test Wilcoxona signed-rank (nieparametryczny odpowiednik t dla prób zależnych)

2. Zmienna porządkowa

→ Test Wilcoxona signed-rank

Założenia

Test t dla prób zależnych

  1. Zależność obserwacji — te same osoby w obu pomiarach (lub pary dopasowane).
  2. Skala co najmniej przedziałowa.
  3. Normalność rozkładu różnic — przy małych n. Przy dużych CTG łagodzi.

Nie ma założenia o homogeniczności wariancji — bo pracujemy na różnicach, nie na dwóch osobnych grupach.

Test Wilcoxona signed-rank

  1. Zależność obserwacji.
  2. Skala co najmniej porządkowa.
  3. Symetryczny rozkład różnic — jeśli niespełnione, rozważ test znakowany (sign test).

Raportowanie (APA 7)

Test t dla prób zależnych

„Lęk istotnie obniżył się po terapii (Mpre = 48.2, SD = 8.3; Mpost = 41.7, SD = 9.1), t(29) = 3.45, p = .002, dz = 0.63."

Wielkość efektu: dz = t / √n (standaryzowane różnicą) lub dav = (M1 − M2) / SDpooled. Podaj, którą formułę stosujesz.

Test Wilcoxona

„Test Wilcoxona wykazał istotną zmianę, W = 45, p = .008, r = .42."

Częste błędy

  1. Testowanie normalności zmiennych zamiast różnic. To rozkład di = x1 − x2 musi być normalny, nie rozkłady x1 i x2 osobno.

  2. Stosowanie testu dla prób niezależnych zamiast zależnych. Jeśli te same osoby → test zależny. Test niezależny ignoruje korelację między pomiarami i traci moc.

  3. Mieszanie dz z d Cohena. W pomiarach powtarzanych istnieją różne formuły d. Zawsze podawaj, którą stosujesz.

  4. Ignorowanie efektu przeniesienia (carryover) w schemacie within-subjects. Jeśli warunki nie są zrównoważone (counterbalancing), różnice mogą wynikać z kolejności, nie z manipulacji.

  5. Raportowanie Mpre i Mpost bez SD i bez statystyk różnic. Potrzebna jest pełna informacja: średnie, SD, t, df, p, d.
