Dwie zmienne kategorialne — jaki test?

Kiedy tego potrzebujesz

Masz dwie zmienne nominalne (kategorialne) i chcesz sprawdzić, czy są od siebie niezależne. Np. czy płeć jest związana z wyborem kierunku studiów? Czy rodzaj terapii wiąże się z wyzdrowienie (tak/nie)?

Algorytm decyzyjny

1. Tabela 2×2 lub większa, wszystkie fe ≥ 5

→ Test χ² Pearsona (chi-kwadrat niezależności)

H0: zmienne są niezależne (rozkład jednej nie zależy od drugiej).

Statystyka χ² porównuje częstości obserwowane (fo) z oczekiwanymi (fe) — oczekiwanymi w przypadku niezależności.

2. Tabela 2×2, jakakolwiek fe < 5 (lub N < 20)

→ Dokładny test Fishera (Fisher’s exact test)

Oblicza dokładne prawdopodobieństwo, nie opiera się na aproksymacji χ².

3. Tabela większa niż 2×2, komórki z fe < 5

→ Opcje:

  • Łączenie kategorii (jeśli merytoryczne sensowne) → χ²
  • Rozszerzony test Fishera (w R: fisher.test())
  • Test Monte Carlo (symulowane p)

Poprawka Yatesa — stosować?

Współcześnie raczej nie. Poprawka na ciągłość jest uznawana za zbyt konserwatywną (zaniża χ², zawyża p). Lepsze rozwiązanie przy małych próbach: dokładny test Fishera.

Progi dla V Craméra zależą od df* = min(wiersze, kolumny) − 1:

  • df* = 1: .10 mały, .30 średni, .50 duży
  • df* = 2: .07 mały, .21 średni, .35 duży
  • df* = 3: .06 mały, .17 średni, .29 duży

Założenia

Test χ² Pearsona

  1. Niezależność obserwacji — każda osoba przyczynia się do jednej komórki tabeli.
  2. Zmienne nominalne (kategorialne).
  3. Dostateczne częstości oczekiwane — wszystkie fe ≥ 5. Jeśli > 20% komórek ma fe < 5 → test Fishera.
  4. Wielkość próby — N ≥ 20 (przy małych N → Fisher).

Test Fishera

Brak założeń o częstościach oczekiwanych — dlatego jest alternatywą przy małych próbach.

Raportowanie (APA 7)

Test χ²

χ²(df, N = [wartość]) = [wartość], p = [wartość], V = [wartość]

Przykład: „Stwierdzono istotny związek między płcią a wyborem kierunku, χ²(2, N = 150) = 8.23, p = .016, V = .23."

Test Fishera

„Dokładny test Fishera wykazał istotny związek między rodzajem terapii a wyzdrowienie, p = .023, φ = .31."

Częste błędy

  1. Stosowanie χ² przy małych częstościach oczekiwanych. Aproksymacja χ² jest niedokładna przy fe < 5. Użyj Fishera.

  2. Raportowanie χ² bez N. Wartość χ² zależy od N — bez podania N jest niekompletna.

  3. Brak miary siły związku. Sama istotność χ² nie mówi o sile. Zawsze podaj φ lub V.

  4. Używanie χ² do danych zależnych. Jeśli pomiary są powtarzane (ta sama osoba w dwóch warunkach) → test McNemara (dla 2×2) lub test Cochrana Q, nie χ² Pearsona.

  5. Mylenie χ² niezależności z χ² dobroci dopasowania. Niezależność: dwie zmienne. Dobroć dopasowania: jedna zmienna vs. oczekiwany rozkład.
