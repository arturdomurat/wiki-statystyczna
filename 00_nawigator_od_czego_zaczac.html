Mam dane — od czego zacząć?

Zanim zaczniesz analizować

Masz zebrane dane i otwierasz program statystyczny. Co teraz? Zanim przejdziesz do testów, wykonaj cztery kroki przygotowawcze:

1. Sprawdź dane — czy nie ma błędów kodowania, braków danych, wartości odstających. → Dane odstające i brakujące — co z nimi robić?

2. Policz statystyki opisowe — średnie, odchylenia standardowe, częstości, wykresy rozkładów. → Statystyka opisowa — co podać i jak? → Wizualizacja danych — wykresy w publikacji

3. Sprawdź rzetelność narzędzi (jeśli używasz kwestionariuszy) — α Cronbacha lub ω McDonalda. → Rzetelność pomiaru — alfa Cronbacha i alternatywy

4. Wybierz analizę — patrz drzewko decyzyjne poniżej.

Jakie jest Twoje pytanie badawcze?

Chcę PORÓWNAĆ GRUPY (czy średnie/mediany się różnią?)

Ile masz grup?

  • Jedna grupa vs. wartość referencyjna → Jedna próba — test jednej średniej
  • Dwie grupy:
    • Niezależne (różne osoby) → Dwie grupy niezależne — jaki test?
    • Zależne (te same osoby, pre-post) → Dwie grupy zależne — jaki test?
  • Trzy lub więcej grup:
    • Niezależne → Więcej niż dwie grupy niezależne — ANOVA
    • Zależne → Więcej niż dwie grupy zależne — ANOVA z powtórzeniami

Dodatkowe komplikacje:

  • Masz więcej niż jeden czynnik? → Schemat czynnikowy — ANOVA wieloczynnikowa
  • Chcesz kontrolować zmienną towarzyszącą? → ANCOVA
  • Masz kilka zmiennych zależnych naraz? → MANOVA

Chcę zbadać ZWIĄZEK między zmiennymi

  • Obie zmienne ciągłe → Korelacja — Pearson, Spearman, Kendall
  • Obie zmienne kategorialne → Zmienne kategorialne — chi-kwadrat, Fisher
  • Chcę PRZEWIDYWAĆ zmienną ciągłą → Regresja liniowa
  • Chcę PRZEWIDYWAĆ zmienną kategorialną → Regresja logistyczna
  • Chcę sprawdzić MECHANIZM (mediacja) lub WARUNEK (moderacja) → Mediacja i moderacja

Chcę zbadać STRUKTURĘ danych

  • Szukam ukrytych wymiarów (nie mam modelu) → EFA — eksploracyjna analiza czynnikowa
  • Chcę potwierdzić model czynnikowy → CFA — konfirmacyjna analiza czynnikowa
  • Chcę testować złożony model teoretyczny → SEM — modelowanie równań strukturalnych
  • Chcę pogrupować osoby w profile → Analiza skupień

Nie wiem, jakiego testu użyć

→ Wybór testu a poziom pomiaru — schematy decyzyjne → Skale Likerta — porządkowe czy przedziałowe?
