Standardy raportowania APA 7

Kiedy tego potrzebujesz

Piszesz wyniki i musisz je sformatować zgodnie z APA 7. Jak zapisać wynik testu t? Ile miejsc po przecinku? Co kursywą? Jak opisać tabelę? Ten wpis to kompaktowy przewodnik.

Pełne zestawienie wzorów: pogotowiestatystyczne.pl/apa-7/

Zasady ogólne

Kursywa

Symbole statystyczne piszemy kursywą: t, F, p, r, d, M, SD, N, n, df, η², b, β, R², U, W, z, χ².

Wyjątek: litery greckie (η², β, χ²) są kursywą, ale ich indeksy nie (η²p).

Zaokrąglanie

  • Dwa miejsca po przecinku dla większości statystyk: t = 2.34, F = 4.56, d = 0.61
  • Trzy miejsca dla p: p = .023
  • Przy p < .001 piszemy: p < .001 (nie podajemy dokładnej wartości)

Zero wiodące

  • Bez zera dla statystyk, które nie mogą przekroczyć 1: p = .023, r = .34, R² = .25, η² = .09
  • Z zerem dla statystyk, które mogą przekroczyć 1: d = 0.61, M = 0.45, b = 0.78

Spacje

Spacje przed i po znaku równości/nierówności: p = .023, nie p=.023.

Wzory raportowania wyników

Test t (Studenta / Welcha)

t(df) = [wartość], p = [wartość], d = [wartość]

Przykłady:

  • t(58) = 2.34, p = .023, d = 0.61
  • t(52.34) = 2.41, p = .019, d = 0.63 (Welch — niecałkowite df)

ANOVA

F(df1, df2) = [wartość], p = [wartość], η²p = [wartość]

Przykłady:

  • F(2, 87) = 4.56, p = .013, η²p = .09
  • F(1.73, 51.84) = 3.21, p = .041, η²p = .07 (z korektą G-G — niecałkowite df)

Korelacja

r(df) = [wartość], p = [wartość] lub r = [wartość], 95% CI [[dolna], [górna]]

Przykłady:

  • r(98) = .34, p < .001
  • r = .34, 95% CI [.16, .50]
  • ρ = .41, p = .003 (Spearman — litera grecka ρ)

Chi-kwadrat

χ²(df, N = [wartość]) = [wartość], p = [wartość], V = [wartość]

Przykład:

  • χ²(2, N = 150) = 8.23, p = .016, V = .23

Test U Manna-Whitneya

U = [wartość], p = [wartość], r = [wartość]

Przykład:

  • U = 312, p = .008, r = .38

Regresja

Model: R² = [wartość], F(df1, df2) = [wartość], p = [wartość]

Pojedynczy predyktor: b = [wartość], SE = [wartość], β = [wartość], t(df) = [wartość], p = [wartość]

Przykład:

  • Model: R² = .34, F(3, 96) = 16.45, p < .001
  • Wiek: b = 0.23, SE = 0.08, β = .27, t(96) = 2.88, p = .005

Tabele (APA 7)

Każda tabela ma:

  1. Numer: Table 1, Table 2… (pogrubiony)
  2. Tytuł: kursywą, pod numerem. Zwięzły, ale informatywny.
  3. Nagłówki kolumn: opisowe, z jednostkami jeśli potrzeba
  4. Linie poziome: górna, pod nagłówkami, dolna. Bez linii pionowych.
  5. Notatka (Note.): pod tabelą, wyjaśnienia skrótów, symbole istotności

Symbole istotności w tabelach: *p < .05. **p < .01. ***p < .001.

Ryciny (APA 7)

Każda rycina ma:

  1. Numer: Figure 1… (pogrubiony)
  2. Tytuł: kursywą
  3. Etykiety osi: pełne nazwy zmiennych (nie skróty)
  4. Legenda: jeśli kilka serii/grup
  5. Słupki błędów: oznaczone (±SD, ±SE, 95% CI) — zawsze podaj, co przedstawiają

Częste błędy

  1. Pisanie p = .000. SPSS zaokrągla do trzech miejsc. Poprawnie: p < .001.

  2. Zero wiodące przy p i r. p = .023, nie p = 0.023. r = .34, nie r = 0.34.

  3. Brak wielkości efektu. APA 7 wymaga ES przy każdym teście. Nie wystarczy podać p.

  4. Brak df przy t i F. t = 2.34 samo w sobie jest bezużyteczne — zawsze t(df) = …

  5. Niespójne formatowanie. W jednym akapicie p = .023, w drugim p = 0,023 z przecinkiem i zerem. Konsekwencja!

  6. Opisywanie wyników słownie zamiast podawania statystyk. „Różnica była istotna" to za mało. Potrzebne: t(df) = …, p = …, d = …

  7. Podawanie p = .05 jako istotne. Przy α = .05 wynik p = .050 formalnie nie jest istotny (p musi być mniejsze niż α, nie równe).
